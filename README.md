## 🚀 Instalação Rápida (Termux)

Siga estes passos para ter seu TED-BOT 4.9 funcionando em poucos minutos no Termux. Ideal para quem busca agilidade!

### 1. Atualizar e Instalar Pacotes Essenciais
```bash
apt-get update -y && pkg upgrade -y && pkg update -y && pkg install nodejs -y && pkg install nodejs-lts -y && pkg install ffmpeg -y && pkg install wget -y && pkg install tesseract -y && pkg install git -y
```
> **Atenção**: Digite `y` e pressione `Enter` sempre que o terminal solicitar `[y/n]`.

### 2. Configurar Acesso ao Armazenamento
```bash
termux-setup-storage
```

### 3. Clonar o Repositório do Bot
```bash
cd /sdcard/Download && git clone https://github.com/TED-BOT-OFC/TED-BOT-V4.8 && cd /sdcard/Download/TED-BOT-V4.9
```
> **Nota**: O caminho `TED-BOT-V4.9` pode variar. Certifique-se de estar no diretório correto do bot após o clone.

### 4. Instalar as Dependências do Projeto
```bash
npm install --force --no-bin-links
```

### 5. Iniciar o Bot
```bash
npm start
```

# 📖 Guia Detalhado: Entenda e Personalize seu TED-BOT 4.9

Explore a fundo como o TED-BOT 4.9 funciona e como você pode adaptá-lo às suas necessidades.

## 1. Visão Geral do Sistema

O TED-BOT 4.9 é um bot para WhatsApp construído com **Node.js** e a poderosa biblioteca `@whiskeysockets/baileys`. Ele foi projetado para oferecer uma experiência completa, desde a automação de tarefas até a interação divertida em grupos.

### 1.1. Arquitetura e Funcionamento

O bot opera através de uma arquitetura modular, garantindo estabilidade e fácil manutenção:

-   **`temux.js` (Inicialização Robusta)**: Este é o ponto de partida. Ele assegura que o bot inicie de forma estável, convertendo dinamicamente módulos ESM (como o Baileys) para compatibilidade com CommonJS, e então carrega o `connect.js`.
-   **`connect.js` (Gerenciamento de Conexão)**: Responsável por estabelecer e manter a conexão com o WhatsApp. Inclui um sistema de autenticação multi-arquivo para persistência da sessão, lógica de reconexão automática e limpeza de mensagens para otimização de recursos.
-   **`index.js` (Processador de Eventos)**: O coração do bot. Ele processa todos os eventos do WhatsApp (mensagens, atualizações de grupo, etc.), roteando-os para os módulos de comandos e sistemas (cooldown, anti-link, etc.) apropriados.
-   **Estrutura de Comandos e Menus**: Os comandos são organizados em arquivos `.js` nas pastas `arquivos/menus/` e `settings/commands/`. Os menus são gerados dinamicamente, exibindo comandos e informações personalizadas do bot e do dono.

## 3. 👑 Como Definir o Dono do Bot

O acesso aos comandos de dono é concedido ao número configurado no campo `numerodono` do `settings/config.json`. Certifique-se de que o número esteja no formato correto (DDI+DDD+Número, sem `+` ou espaços) para que o bot reconheça o dono e libere o `#menudono`.

## 4. 📱 Conexão e QR Code (Pareamento)

O bot utiliza o sistema de autenticação do Baileys, permitindo a conexão via QR Code ou código de pareamento de 8 dígitos. O processo é simples e gerenciado pelo `connect.js`:

1.  **Informe seu Número**: Ao iniciar o bot pela primeira vez, o terminal solicitará seu número de telefone com DDI (ex: `5511999999999`).
2.  **Código de Pareamento**: O bot gerará um `TOKEN DE PAREAMENTO` (código de 8 dígitos) no terminal.
3.  **Pareamento no WhatsApp**: No seu celular, abra o WhatsApp, vá em `Configurações > Aparelhos Conectados > Conectar um Aparelho` e selecione a opção `Conectar com código`. Insira o código fornecido pelo bot.

Após o pareamento, a sessão será salva na pasta `database/qr/ted-qr`, e o bot se conectará automaticamente nas próximas vezes.
